# Intelligence embarquée

Ici se trouvera le code de l'automate de stratégie du robot qui sera embarqué
dans une carte de type Raspberry Pi.

## Install dependencies

```sh
poetry install
```

## Run the automatus

```sh
poetry run main.py
```
